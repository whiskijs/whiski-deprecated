# @whiski/cli

Command line interface for whiski.js

## installation

```bash
$yarn global add @whiski/cli

$npm install -g @whiski/cli
```

## usage

```bash
$whiskir https://cdn.jsdelivr.net/npm/lodash@4.17.21/lodash.min.js
```

install package with options:

```bash
$whiskir <URL> --out=output
```
